Projet : Affichage de la météo dans les transports en commun
Version 1

1. Le fichier conf.json contient la ville pour laquelle la météo est affichée.
2. Le script meteo.js récupère les données depuis OpenWeatherMap.
3. L'affichage se fait dans index.html, stylisé avec style.css.
4. Les données sont mises à jour toutes les heures en rechargeant la page.
